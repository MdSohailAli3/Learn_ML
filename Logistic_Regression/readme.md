# Student Placement Prediction

This is a simple Machine Learning project that predicts whether a student will be placed based on **CGPA** and **IQ**.

The model is built using **Logistic Regression** with the Scikit-learn library.

## Workflow
- Load dataset
- Data visualization
- Train-test split
- Feature scaling
- Train Logistic Regression model
- Evaluate model accuracy

## Technologies
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Purpose
This project was created for learning and practicing basic Machine Learning concepts.
