# Machine Learning Learning Projects

This repository contains small Machine Learning projects and experiments created while learning different ML algorithms and concepts.

Each folder in this repository focuses on implementing and understanding a specific Machine Learning technique using Python.

---

## Purpose of This Repository

The goal of this repository is to:

- Practice Machine Learning concepts
- Implement algorithms using Python and Scikit-learn
- Understand data preprocessing and model training
- Build a structured ML learning portfolio

---

## Projects Included
